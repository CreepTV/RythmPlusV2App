 **Rhythm Plus V2 – Unofficial Web App**



# 🐧 v1.1.7 — Linux is here!

## Highlight

**Rhythm Plus V2 is now available on Linux!**
You can now download the app as an **AppImage** and run it on popular Linux distributions without installation.

---

## What's new in version v1.1.7:

- **Linux support** — The app is now available as a Linux (x64) AppImage.
- **Load URLs directly** — You can now paste a Rhythm Plus URL from the title bar and open it immediately in the app.
- **Switch between v1 and v2** — In Settings, you can choose Rhythm Plus v1 (rhythm-plus.com) or v2 (v2.rhythm-plus.com).
- **New in Settings:** — You can now switch directly between **Rhythm Plus v1** and **Rhythm Plus v2**.

## Improvements

- Login and authentication are now more reliable, reducing sign-in issues.
- Closing the preview window in the editor now properly stops music playback.
- Automatic update behavior has been improved.

## Fixes

- Fixed rare navigation issues in game preview windows.
